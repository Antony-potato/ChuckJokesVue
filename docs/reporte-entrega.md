Descripcion

La presente actividad consiste en desarrollar una aplicacion con Vue que permita mostrar un conjunto de chistes de Chuck Norris a partir de un arreglo de objetos. La informacion se carga desde el componente principal y se presenta en pantalla mediante una lista renderizada dinamicamente con la directiva v-for. Ademas, se aplico un estilo visual minimalista para mejorar la presentacion del contenido.

Objetivo

Trabajar con el framework JavaScript Vue para construir una interfaz funcional, clara y agradable, usando datos dinamicos y una estructura moderna de proyecto basada en Vite.

Procedimiento realizado paso a paso

1. Se exploro el workspace para verificar si existia un proyecto previo y confirmar la disponibilidad de Node.js, npm y Git.
2. Se creo un proyecto nuevo llamado ChuckJokesVue utilizando Vue 3 con Vite.
3. Se instalaron las dependencias necesarias del proyecto para su ejecucion local.
4. Se sustituyo la plantilla inicial por una vista principal personalizada.
5. Se agrego el arreglo de chistes de Chuck Norris dentro del componente principal App.vue.
6. Se utilizo la directiva v-for para recorrer el arreglo y mostrar cada chiste en una tarjeta visual.
7. Se diseno una interfaz minimalista con CSS, cuidando colores, tipografia, espaciado y adaptacion a pantallas pequenas.
8. Se actualizo el archivo README.md con la descripcion de la actividad, el objetivo, las tecnologias usadas, los comandos para ejecutar el proyecto y la URL prevista del repositorio.
9. Se inicializo Git en el proyecto para dejarlo listo para control de versiones.
10. Se genero este reporte en formato editable y en formato Word para su entrega.
11. Se valido el proyecto mediante la compilacion de produccion para comprobar que la aplicacion funciona correctamente.
