# Actividad 12: Chuck Jokes Vue Responsive Layout

Proyecto desarrollado con **Vue 3**, **Vite** y **Bootstrap 5** para mostrar una colección de chistes de Chuck Norris utilizando componentes reutilizables, pasaje de props y diseño responsivo.

## Descripción de la actividad

En esta actividad se actualizó la estructura de datos para incluir una URL de imagen (`icon_url`) junto con el texto del chiste. Se implementó un componente especializado llamado `chuck-card` para renderizar la información y se utilizó el sistema de grid de Bootstrap para asegurar que la aplicación sea responsiva.

## Objetivos

- Trabajar con componentes de Vue y pasaje de datos mediante **props**.
- Implementar la directiva `v-for` para renderizado dinámico de listas.
- Aplicar diseño responsivo utilizando el sistema de rejilla (Grid) de **Bootstrap**.
- Crear una interfaz estética, minimalista y profesional.

## Tecnologías usadas

- **Vue 3** (Composition API)
- **Vite** (Build Tool)
- **Bootstrap 5** (Layout y Estilos)
- **JavaScript** (ES6+)
- **CSS3** (Personalización Premium)

## Estructura del Layout

- **Desktop**: 3 tarjetas por fila.
- **Mobile**: 1 tarjeta por fila.
- **Detalle**: En el caso de 5 elementos, la última fila muestra 2 tarjetas centradas para mantener el equilibrio visual.

## Instrucciones para correr el proyecto

1. Abrir una terminal en la carpeta `ChuckJokesVue`.
2. Instalar dependencias:
   ```bash
   npm install
   ```
3. Ejecutar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Navegar a: `http://localhost:5173`
