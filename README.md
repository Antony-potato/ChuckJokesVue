# ChuckJokesVue

Proyecto desarrollado con Vue para mostrar una coleccion de chistes de Chuck Norris usando datos definidos en el componente principal y renderizados con la directiva `v-for`.

## Descripcion de la actividad

La actividad consiste en construir una aplicacion con Vue que tome un arreglo de objetos con chistes de Chuck Norris y los muestre en pantalla con una vista agradable, ordenada y facil de leer.

## Objetivo

Trabajar con el framework JavaScript Vue para crear una vista principal que renderice datos dinamicos y aplique estilos personalizados a la interfaz.

## Tecnologias usadas

- Vue 3
- Vite
- JavaScript
- CSS
- Git

## Instrucciones para correr el proyecto

1. Abrir una terminal en la carpeta del proyecto.
2. Instalar dependencias con:

```bash
npm install
```

3. Ejecutar el servidor de desarrollo con:

```bash
npm run dev
```

4. Abrir en el navegador la URL local que muestre Vite, normalmente `http://localhost:5173`.

## Validacion de produccion

Para generar la version de produccion:

```bash
npm run build
```
